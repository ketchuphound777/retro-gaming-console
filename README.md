Retro Gaming Console
This folder contains all documentation, photos, and videos for my custom retro gaming console built using an UP Board. The goal of this project was to create a compact system capable of running classic games while learning more about hardware integration, Linux configuration, and system setup on x86‑based embedded hardware.

About This Folder
I no longer have the original configuration files, scripts, or system images for this project. Because of that, this folder includes:

photos of the hardware, wiring, and assembly

videos showing the system running

notes describing the setup process

documentation of the hardware and software environment

This repository serves as a record of the project even without the original files.
